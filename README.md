# 2. Definição do Projeto

## 2.1 Visão

Desenvolver uma plataforma digital que conecte voluntários a ações sustentáveis, permitindo o registro e acompanhamento de atividades ecológicas, com um sistema de pontuação baseado em “Créditos Verdes”.

O sistema permitirá que usuários acompanhem seu impacto ambiental, enquanto administradores poderão gerenciar dados e gerar relatórios.

---

## 2.2 Objetivos

**Objetivo SMART:**

Desenvolver, em até 4 meses, um sistema backend em Java com banco de dados relacional integrado, capaz de cadastrar usuários, armazenar atividades sustentáveis e calcular automaticamente créditos verdes, garantindo segurança de dados e suporte a pelo menos 500 usuários em ambiente de teste.
